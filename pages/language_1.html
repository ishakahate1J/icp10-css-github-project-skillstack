Java Tutorial

What is Java?

Java is a programming language and a platform. Java is a high-level, robust, object-oriented, and secure programming language.
Java was developed by Sun Microsystems in 1995, and James Gosling is known as its father. Initially called Oak, the name was later changed to Java

Java Data Types

Java data types are categorized into:

  • Integer Types: Stores whole numbers (e.g., byte, short, int, long).
  • Floating Point Types: Represents numbers with decimals (e.g., float, double).

Syntax:


            int myNum = 5;               // Integer (whole number)
            float myFloatNum = 5.99f;    // Floating point number
            boolean myBool = true;       // Boolean
            char myLetter = 'D';         // Character
            String myText = "Hello";     // String

Java Operators

            Java divides the operators into the following groups:

            1)Arithmetic operators

            2)Assignment operators

            3)Comparison operators

            4)Logical operators

            5)Bitwise operators

Example

                
                public class OperatorExample {
                public static void main(String[] args) {
                    int a = 10, b = 5;
                    // Arithmetic
                    System.out.println("a + b: " + (a + b)); // 15
                    // Assignment
                    int c = a;
                    c += b;
                    System.out.println("c after += b: " + c); // 15
                    // Comparison
                    System.out.println("a > b: " + (a > b)); // true
                    // Logical
                    System.out.println("(a > b) && (b > 0): " + ((a > b) && (b > 0))); // true
                    // Bitwise
                    System.out.println("a & b: " + (a & b)); // 0
                }
            }

Java Looping

Java provides the following loops to perform repetitive tasks:

  • For Loop: Repeats a block of code a specific number of times. 
    public class ForExample {  
                public static void main(String[] args) {  
                    for(int i=1;i<=10;i++){  
                        System.out.println(i);  
                    }  
                }  
                }
  • While Loop: Repeats a block of code while a condition is true. 
                    
                int i = 0;
                while (i < 5) {
                  System.out.println(i);
                  i++;
                }
  • Do-While Loop: Executes a block of code at least once, then repeats it while a condition is true. 
                    int i = 0;
                do {
                  System.out.println(i);
                  i++;
                }
                while (i < 5);

Java Methods

A Java method is a block of code that performs a specific task:

  • Syntax: returnType methodName(parameters) { // code }
  • Types: Predefined methods (e.g., System.out.println()) and user-defined methods.

    Example

                    public class Main {
                static void myMethod() {
                  System.out.println("I just got executed!");
                }
                public static void main(String[] args) {
                  myMethod();
                }
              }
              // Outputs "I just got executed!"

Java Classes

A class is a blueprint for creating objects in Java:

  • Definition: class ClassName { // fields and methods }
  • Classes encapsulate data and methods together.
  • Objects are instances of classes.

Example

            class ClassName {
            // Fields (attributes)
            String name;
            int age;
        
            // Constructor
            ClassName(String name, int age) {
                this.name = name;
                this.age = age;
            }
        
            // Method
            void displayInfo() {
                System.out.println("Name: " + name + ", Age: " + age);
            }
        }

Java File Handling

Java provides classes for reading and writing files, such as:

  • FileReader: Reads data from a file.
  • FileWriter: Writes data to a file.
  • BufferedReader: Reads text from an input stream efficiently.

Example:

import java.io.File;  
public class FileExample {  
    public static void main(String[] args) {  
        File file = new File("example.txt");  
        System.out.println("File exists: " + file.exists());  
    }  
}
